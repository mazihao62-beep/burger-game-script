# 🍔 汉堡游戏自动脚本

## 使用方法

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/mazihao62-beep/burger-game-script/main/burger_script.lua"))()
```

## 功能

| 功能 | 说明 |
|------|------|
| ⚔️ **自动杀死NPC** | 装备铲子自动攻击附近NPC |
| 🧹 **自动粉碎尸体** | 用麻袋捡尸体放入粉碎机 |
| 🍔 **自动做汉堡** | 自动拾取食材制作汉堡 |
| 💰 **自动收集金钱** | 自动传送到金钱位置收集 |
| 🤖 **全自动模式** | 一键启动所有自动化流程 |
| 👁 **NPC透视** | Highlight + 头顶标签 |

## 操作

| 平台 | 方式 |
|------|------|
| 💻 PC | 按 **RightShift** 打开菜单 |
| 📱 手机 | 点击悬浮按钮 |
